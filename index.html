HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1,h2,h3,h4 etc.. represet the headers.
  • Unordered lists, ordered lists, listed items are abbreviated as follows: ul, ol, li.
  • Any elements held within side another element are knownn as child elements.
  • The element that is holding the other element is knownn as the parent element.
  • Section elements are semantic elements that define a section in a document that contains thematic content. Ex HTML, CSS, Git, JavaScript.
  • We should use Semantic HTML elements such as HTML whenever we can because they make yopur HTML easier to read by other developers and the browser.
  • In Semantic HTML elements the name of the element describes its content and the role it plays on the webpage.
  • When working with Semantic HTML elements, the content is cointained between and openeing and closing tag represented by angle brackets.
  • These are often associated with tag names contained inside the brackets.
  • Closing tags begin with a forward slash.

CSS

  • CSS Stands for Cascading Style Sheets
  • CSS is what makes your webpage "look good"
  • CSS allows the user to easily find and read the information
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is essentially the same thing as github
  • git status: checks what branch we are currently on
  • git checkout -b bramch name: creates a new branch and switches to it

JavaScript

  • JavaScript is how we add logic to a webpage
  • JavaScript allows a user to interat with a webpage
  • Using JavaScript, you can power your websites with effects, interactivity, data communication, and much more.
  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
